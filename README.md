# HealthyBuddy

HealthyBuddy is a health monitoring application prototype that supports
SDG 3: Good Health and Well-Being.

## Features
- Daily health tracking (food, water, sleep, exercise)
- Mental health check-ins
- Reminders and notifications
- Progress tracking with charts
- Buddy system for motivation

## Target Users
- Students
- Young adults
- Community members

## Tools Used
- Figma (UI/UX Prototype)
- GitHub (Documentation)

## SDG Alignment
This project supports SDG 3 by promoting healthy habits,
mental well-being, and self-monitoring.
